Grid

O sistema de grid padrão utiliza 16 colunas, cada coluna com 40px de largura e 20px de espaçamento entre elas, totalizando 940px.

Basta usar as classes span1...16, dentro de uma .row, para escolher as larguras dos blocos.

Arquivos:
  • _variables.scss: configurações de largura de coluna, quantidade e espaçamento;
  • _mixins.scss: funções que geram o sistema de grid;
  • _grid.scss: chamada da função.
a.v.a.s, cursos, disciplinas, aulas, perfis
  <div class="row">
    <div class="span3"></div>
    <div class="span13">a.v.a.s, cursos, disciplinas, aulas, perfis</div>
  </div>
início
  <div class="row">
    <div class="span3"></div>
    <div class="span10">início</div>
    <div class="span3"></div>
  </div>

Deslocando as colunas

Para deslocar as colunas de posição, basta usar as classes offset1...15.

4 6 offset 6 
  <div class="row">
    <span class="span4">4</span>
    <span class="span6 offset6">6 offset 6</span>
  </div>
3 offset 3 3 offset 3 
  <div class="row">
    <span class="span3 offset3">3 offset 3</span>
    <span class="span3 offset3">3 offset 3</span>
  </div>
13 offset 3 
  <div class="row">
    <span class="span13 offset3">13 offset 3</span>
  </div>
