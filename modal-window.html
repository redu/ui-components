Janela modal

Você pode ativar as janelas modais na página sem a necessidade de escrever JavaScript. Apenas coloque data-toggle="modal" no elemento de controle e href="#foo" que corresponde ao id da janela modal, e quando clicado, ele irá abrir a janela modal.

  • Clique aqui para abrir o exemplo 1 (só texto).
  • Clique aqui para abrir o exemplo 2 (vídeo do YouTube).
  • Clique aqui para abrir o exemplo 3 (muito texto).
Arquivo

_modal-window.scss

Dependência

bootstrap-modal.js

  <a data-toggle="modal" href="#MODAL_ID">Clique aqui</a>

  <div class="modal hide fade" id="MODAL_ID">
    <div class="modal-header">
      <h3 class="modal-header-title">MODAL_TITLE</h3>
      <!-- IF MODAL_HAS_SUBTITLE, DOESN'T NEED TO BE LINK BUT MUST HAVE THE CLASS modal-header-subtitle -->
      <a class="modal-header-subtitle" href="MODAL_SUBTITLE_LINK" title="MODAL_SUBTITLE">MODAL_SUBTITLE</a>
      <!-- END -->
    </div>
    <div class="modal-body">
      MODAL_CONTENT
    </div>
    <div class="modal-footer">
      <!-- NEED TO HAVE THE ATTRIBUTE data-dismiss="modal" TO BE ABLE TO CLOSE THE MODAL -->
      <button data-dismiss="modal" class="button-primary" title="Fechar">Fechar</button>
    </div>
  </div>
